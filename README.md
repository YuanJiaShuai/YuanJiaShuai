## Featured Projects

### [tdx-api](https://github.com/YuanJiaShuai/tdx-api)

基于通达信协议的股票行情数据服务，提供 REST API、Web 可视化、公式选股、自动化任务与 Docker 部署。

### [YJSScrollMenu](https://github.com/YuanJiaShuai/YJSScrollMenu)

一款类似美团、淘宝商城首页的 icon menu 菜单组件，适合移动端首页分类入口场景。

### [YJSStockKeyboardView](https://github.com/YuanJiaShuai/YJSStockKeyboardView)

适用于股票应用场景的自定义键盘组件，用于提升证券、行情、交易类 App 的输入体验。

### [speaknest](https://github.com/YuanJiaShuai/speaknest)

语言学习相关产品探索，围绕表达、练习和学习效率构建更轻量的工具体验。
